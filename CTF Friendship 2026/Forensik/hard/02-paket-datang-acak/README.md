# Paket Datang Acak

**Kategori:** Advanced Network Forensics  
**Kesulitan:** Hard  
**Artifact:** `mixed\\\\\\\_capture.pcap`

Malware mengirim hasil incident response ke luar jaringan. Payload TCP tiba
tidak berurutan dan tampak seperti sampah. Tepat sebelumnya, host menerima
beberapa jawaban DNS yang tidak biasa.

Rekonstruksi payload dan temukan flag.

**Hint:** TTL bukan selalu sekadar umur, dan urutan tangkap bukan selalu
urutan kirim.



flag : FS26{wireshark\_saksi\_bisu\_semua\_dosamu\_di\_jaringan}
